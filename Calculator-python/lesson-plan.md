# Python Programming Lesson Plan for Grade 11

## Lesson Objective
Students will learn basic Python programming concepts including functions, user input, and conditional statements through practical examples.

## Duration
90 minutes

## Materials
- Computers with Python installed
- main.py file as a reference

## Lesson Outline

1. Introduction (10 minutes)
   - Brief overview of Python
   - Importance of functions in programming

2. Basic Function Structure (15 minutes)
   - Explain the syntax of defining a function
   - Show examples from main.py (addition, subtraction, multiply, division functions)

3. User Input (15 minutes)
   - Demonstrate how to get user input using input() function
   - Convert input to appropriate data types (int, float)

4. Conditional Statements (20 minutes)
   - Introduce if, elif, and else statements
   - Explain how to use comparison operators

5. Hands-on Activity: Calculator Program (25 minutes)
   - Guide students to create a simple calculator program similar to the one in main.py
   - Encourage them to add their own functions or modify existing ones

6. Wrap-up and Discussion (5 minutes)
   - Recap key concepts learned
   - Answer any questions

## Extension Activities
1. Implement the commented-out weather reporter function
2. Create a vowel counting program as shown in the comments

## Assessment
- Students will be evaluated based on their participation and the functionality of their calculator program
- Bonus points for implementing extension activities

## Homework
Modify the calculator program to include additional mathematical operations (e.g., exponentiation, square root)
